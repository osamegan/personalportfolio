The software I used was Visual Studio Code and
the language was CSS and JavaScript. Some challenges my partner and I faced during this project mainly
had to do with getting everything working as well as looking good. The first challenge we faced was

very well. A concept I learned was that you can never really be done. There is always something else
you can improve on or fix.
